RP2040 Pizza Detection System - Gerber Files für JLCPCB
===========================================

Diese Gerber-Dateien wurden für die Fertigung bei JLCPCB optimiert.
Sie entsprechen den JLCPCB-Anforderungen vom Mai 2025.

Folgende Dateien sind enthalten:
- PizzaBoard-RP2040.GTL: Top Layer
- PizzaBoard-RP2040.GBL: Bottom Layer
- PizzaBoard-RP2040.GTO: Top Silk Screen
- PizzaBoard-RP2040.GBO: Bottom Silk Screen
- PizzaBoard-RP2040.GTS: Top Solder Mask
- PizzaBoard-RP2040.GBS: Bottom Solder Mask
- PizzaBoard-RP2040.GTP: Top Paste
- PizzaBoard-RP2040.GBP: Bottom Paste
- PizzaBoard-RP2040.GKO: Board Outline
- PizzaBoard-RP2040.TXT: NC Drill File

PCB-Spezifikationen:
- Layer: 8
- Dicke: 1.6mm
- Kupferstärke: 1oz
- Min. Leiterbahnbreite/Abstand: 0.15mm/0.15mm
- Min. Bohrungsdurchmesser: 0.3mm
- PCB-Farbe: Grün
- Oberflächenveredelung: HASL mit Blei
