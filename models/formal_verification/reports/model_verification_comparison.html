Notes on Formal Verification

Formal verification provides mathematical guarantees that neural networks satisfy certain properties, unlike traditional testing which only checks specific inputs. The verification properties tested include:

  • Robustness: The model maintains the same prediction when inputs are perturbed slightly.
  • Brightness Invariance: The model gives consistent predictions across different brightness levels.
  • Class Separation: Critical classes (e.g., raw vs. burnt) are never confused.
